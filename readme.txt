----------------------Install----------------------
Nothing Just Run skid.exe

-------------------Configuration-------------------
Configuration config.json file:
prefix: your prefix you want eg: {prefix}help
...

------------------------Notes----------------------
- this is not a tool, this is a selfbot, use commands in discord
- By default, mass mentions will be turned off
  If you want to turn it on, open the settings.json file
- guild_id is server_id
{prefix}help for show all commands
{prefix}help "command" for show usage instructions

----------------------Commands---------------------
replace your prefix. my prefix eg: "."
Console Commands: 
{prefix}stop: to stop all thread eg: spam, dm, ...
{prefix}clear or {prefix}cls: to clear console

----------------------balance----------------------
{prefix}getethbal "eth address"
{prefix}getbtcbal "btc address"
{prefix}getltcbal "ltc address"
{prefix}addy: show all address
{prefix}btc: show btc address
{prefix}eth: show eth address
{prefix}ltc: show eth address
{prefix}calc

---------------------------------------------------

If there is a space, put it in " " eg: {prefix}spam "Skid Raider On Top"

{prefix}join: Join all tokens to the server
eg: you can type only "{prefix}join" if you are on a channel of the target server
or you can type: {prefix}join "invite"

{prefix}leave: Leave all tokens in the server
eg: you can type only ({prefix}leave) if you are on a channel of the target server
or you can type ({prefix}join "server_id - guild_id")

{prefix}spam: Spam channel:
eg: you can type only ({prefix}spam "message" "delay") if you are on a channel of the target server,
you can type: {prefix}spam "message" (default: delay=0)
or you can type: {prefix}spam "message" "delay" "channel_id" "guild_id"
you can set delay = 0

{prefix}poll: Spam poll:
eg: {prefix}poll or {prefix}poll "channel_id" "delay"
you can type: {prefix}poll "channel_id" (default: delay=0)
note: you need configuration in settings{prefix}json

{prefix}nick: Nick all tokens in the server
eg: you can type only ({prefix}nick "your name you want") if you are on a channel of the target server
or you can type ({prefix}nick "your name you want" "server_id - guild_id")

{prefix}vc: Voice call joiner
eg: {prefix}vc "channel_id" "guild_id"

{prefix}reaction: add reaction to message
eg: you can type: {prefix}reaction "ðŸ’Ž" "channel_id" "message_id" 
or you can type: {prefix}reaction "ðŸ’Ž" "messagelink": {prefix}reaction "ðŸ’Ž" "https://discord.com/channels/1187421717686124735/1188475786093670562/1225140443453456447"

{prefix}button: ({prefix}button "button number, starting from 1" "channel_id" "guild_id" "message_id")
or {prefix}button "button number, starting from 1" "messagelink"

{prefix}check: check all tokens
{prefix}boost: (same) use {prefix}boost or {prefix}boost "guild_id"
{prefix}typing: (same) use {prefix}typing or {prefix}typing "channel_id"
{prefix}vaultcord "botid": bypass vaultcord
{prefix}message "message" "channelid": spam message onetime

{prefix}call: spam calling
eg: {prefix}call "your target userid"

{prefix}friend: spam friend

{prefix}unfriend: ...
